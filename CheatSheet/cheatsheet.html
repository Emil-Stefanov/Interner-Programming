HTML Cheatsheet

A handly reference for important HTML tags

Documenet Structure & Metadata Tags

<html> Определя началото и края на HTML документа.
<head> Съдържа мета информация за документа (напр. заглавие, стилове).
<title> Определя заглавието на страницата, което се показва в браузъра.
<meta> Предоставя мета информация за документа (ключови думи, описание).
<link> Използва се за свързване на външни ресурси като CSS файлове.
<style> Съдържа CSS стилове, които форматират HTML елементите.
<script> Включва JavaScript код за динамично поведение на страницата.

Layuot/Sectioning Tags

<body> Съдържа видимото съдържание на уеб страницата.
<header> Представя хедър (заглавна част) на страницата или секция.
<footer> Представя футър (долна част) на страницата или секция.
<section> Разделя съдържанието на отделни секции.
<article> Определя независим елемент от съдържанието, например статия.
<nav> Съдържа навигационни линкове.
<aside> Съдържа странична информация, често в странична лента.
<main> Определя основното съдържание на страницата.
<div> Контейнер за групиране на HTML елементи.

Text Content & Formatting Tags

<h1> - <h6> Определят заглавия с различни нива на важност.
<p> Определя абзац текст.
<strong> Показва текста с удебелено значение (силно значение).
<em> Подчертава текста с курсив за акцент (емфаза).
<b> Удебелява текста.
<i> Прави текста курсивен.
<span> Контейнер за вътрешно групиране и форматиране на малки елементи.
<br> Добавя нов ред в текста.

Hyperlink & Media Embedding Tags

<a> Създава хипервръзка (линк).
<img> Вмъква изображение.
<audio> Вмъква аудио файл.
<video> Вмъква видео файл.
<iframe> Вгражда съдържание от друга уеб страница (напр. видео от YouTube).

List Tags

<ul> Определя неномериран (булетен) списък.
<ol> Определя номериран списък.
<li> Представлява елемент от списък.

Table Tags

<table> Създава таблица.
<tr> Определя ред в таблица.
<td> Създава клетка в таблица (данни).
<th> Създава заглавна клетка (обикновено удебелена).
<thead> Определя хедър частта на таблица.
<tbody> Определя основната част от съдържанието на таблицата.
<tfoot> Определя футър частта на таблицата (крайни резултати или забележки).

Form & Input Tags

<form> Създава форма за събиране на потребителски данни.
<input> Определя полета за въвеждане на данни.
<textarea> Създава поле за много редове текст.
<button> Създава бутон.
<select> Създава падащо меню за избор.
<option> Определя опция в падащо меню.
<label> Определя опция в падащо меню.

Media & Figure Tags

<figure> Представя мултимедийно съдържание (снимка, диаграма).
<figcaption> Добавя описание към мултимедийното съдържание.

Interactive Elements & Special Purpose Tags

<details> Създава разгъваем раздел, който показва/скрива съдържание.
<summary> Заглавие на разгъваемия раздел.
<progress> Представя лента за прогрес (например при зареждане).
<meter> Показва измерване на стойност в диапазон (например резултат).

Miscellanous & Separator Tags

<hr> Добавя хоризонтална линия, която разделя съдържанието.
